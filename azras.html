ADAPTIVE ZERO-REBUILD ASSET SYSTEM
AZRAS
Adaptive Zero-Rebuild Asset System
"Build Once. Value Endures."
Zero rebuild. Value for 100 years ahead. A sustainable asset system for the future.

AZRASは、永続的に耐久するRC構造コアと、住戸単位で独立交換可能な木造インフィルを融合した次世代ハイブリッド建築システムです。構造とインフィルを分離することで、建物生涯にわたる全面建て替えを不要にし、世代を超えた建築資産の価値を実現します。

開発：赤津 牧人 ／ 株式会社ACE総合コンサル（旧・有限会社赤津設計）　2005年 日本にて実棟竣工。

CONCEPT

Open-Core Architecture の設計思想

AZRASの核心は「構造とインフィルの完全分離」です。永続するRC構造コアが建物を支え続ける一方、木造インフィルは時代のニーズに合わせて自由に更新されます。この分離が、建て替えゼロ・100年以上の資産価値維持を実現します。

ADAPTIVE INFILL — 交換可能部
木造インフィル（自由に更新）
各住戸内の木造部分は、隣戸の居住状態や構造コアに影響を与えることなく、住戸単位で独立して全面交換が可能。時代や用途の変化に柔軟に対応します。2回目以降の更新費用は新築時の約半額を実現。
PERMANENT CORE — RC構造コア
RC構造コア（100年以上の耐久性）
ベタ基礎・一方向の耐力外壁・住戸間の界壁で構成されるRC構造コアは、100年以上の耐久性を前提に設計。一般的なSI（スケルトン・インフィル）工法と異なり、外壁面にRCの梁がないため、更新時に外観の全面刷新も可能。
ADAPTIVE INFILL（木造インフィル）
時代と用途に合わせて自由に更新可能。住戸単位での独立交換により、隣戸への影響なし。上階全体が木造のため、間取り・開口・空間レイアウトを次世代に応じて変更可能。
PERMANENT CORE（RC構造コア）
100年以上の耐久性。建物生涯を通じて建て替えゼロを実現し、長期にわたって資産価値を維持・継承。津波・洪水時も2階以上での即時事業継続が可能。
KEY FEATURES

AZRASの6つの特長

01
LOW COST · HIGH DURABILITY
低コスト・高耐久の両立
2005年実績において、初期建設コストを軽量鉄骨造並みに抑えながら、RCコアは100年以上の耐久性を確保。2回目以降の更新費は新築時の約半額。一般的なRC建築を大幅に下回るコストを実現しました。
02
UNIT-BY-UNIT RENEWAL
住戸単位での独立更新
退去が発生した住戸から順次、隣戸の居住・営業状態を維持したまま木造インフィルを全面交換可能。「建物は区分所有・土地は単有」の登記構造により、相続時の柔軟な資産分割と住戸単位での部分売却にも対応。
03
SEISMIC PERFORMANCE
実証済みの耐震性能
2005年に外部専門機関による動的非線形地震応答解析（6月22日）と、ビイック株式会社（VIIC）による実棟振動応答特性計測（5月18日）の2つの独立した方法で耐震安全性を実証済み。
04
DISASTER RESILIENCE
災害レジリエンス
津波・洪水等で1階の木造部分が損傷した場合でも、RCコアが2階床・屋根を支持し続けるため、2階での即時避難生活・事業継続が可能。復旧は損傷した木造部分のみの改修で完了し、全体の建て替えは不要。
05
ASSET VALUE
資産価値の維持・向上
建て替えなしで資産価値を維持・向上。2005年実績では¥110,000/戸（周辺相場比+83%超）の賃料を達成。月額総収益¥330,000対 市場相場¥240,000（37.5%増）を実現しました。
06
ENERGY EFFICIENCY
省エネルギー性能
外断熱とコンクリートの蓄熱性を組み合わせることで、建物の生涯を通じて冷暖房負荷を継続的に低減。長期的な運用コストの削減と環境負荷の軽減を両立します。
FUTURE-ADAPTIVE LIVING

可変性のある未来の住まい

AZRASの最大の特長は「用途が変わっても建て替え不要」であること。同じRC構造コアを維持しながら、木造インフィルを交換するだけで建物の用途・デザインを世代を超えて変え続けることができます。

STAGE 1
南向きウッドデッキのある住宅
南側に広がるウッドデッキが室内と屋外をつなぎ、明るく開放的なリビングと庭を実現。家族の暮らしを豊かに育む住まい。
STAGE 2
イタリアンレストランへ転用
大きく開いた南側の開口部が街とつながる心地よい空間に。木造インフィルの交換だけで完全に用途を転換。建て替え不要。
STAGE 3
1階が車庫・2階が先進的な住宅へ
ライフスタイルの変化に合わせて機能と価値を最適化。長く快適に住み続けられる次世代の住まいへと進化。
FLEXIBLE SPACES
ライフスタイルや用途の変化に合わせて、間取りや使い方を自由に変更可能。
SCALABLE STRUCTURE
将来の増築や用途変更にも対応できる、柔軟で強靭な構造設計。
SUSTAINABLE DESIGN
環境負荷を抑え、長く使い続けることができるサステナブルな建築。
MAXIMIZED ASSET VALUE
時代やニーズに合わせて価値を高め続ける、資産性の高い建築。

INTERNATIONAL EXPANSION

国際展開

AZRASの構造的論理は特定の工法に依存せず、各国の構造基準・建設慣行に適応できるフレームワークです。

JAPAN — 国内展開
日本国内での展開
2005年の竣工後、日本の構造審査基準が大幅に厳格化されました。現行基準での新築申請には、通常の確認申請に加えて専門機関による特別な構造評定・大臣認定の取得が必要となる見込みですが、相応の審査期間と費用をもって対応可能です。商用ライセンス供与・技術監理・設計サポートを提供します。
商用ライセンス 技術監理 構造評定支援
INTERNATIONAL — 海外展開
海外市場への構造的適応
オリジナルシステムのRC壁を、RC壁をモーメントフレーム（柱と梁）に置き換えることで、米国・東南アジア・中東・中南米など異なる構造規範や耐震要件を持つ市場にも対応可能。「耐久性のある構造シェル＋交換可能インフィル」というコアコンセプトを維持しながら現地基準に適合。
